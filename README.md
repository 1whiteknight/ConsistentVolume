# ConsistentVolume
A small program for normalizing different audio files to the same volume level, making it easier to edit audio and video files later on.
Created with ChatGPT, based on my idea.
Written in Python.

1. Chose the file you need.
2. Chose the folder where to seve the new file.
3. Press button togenerate the new file.

What you still need to do (in Windows)
Install Python (if you don't already have it)
Install ffmpeg
Important: ffmpeg must be in your PATH (add ffmpeg.exe to the PATH)
or located in the same folder as the Python script
